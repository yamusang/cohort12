## 오늘의 핵심
- interrupts:  특정 지점에서 그래프 실행을 일시 중지하고 외부 입력을 기다린 후 다시 실행할 수 있도록 하는 기능으로, 
               인터럽트가 발생하면 LangGraph는 영구 저장 계층을 사용하여 그래프 상태를 저장하고 사용자가 실행을 재개할 때까지 무기한 대기한다.
               그리고 고객의 값을 받아 노드를 처음부터 실행한다.
