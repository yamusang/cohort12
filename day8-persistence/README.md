# day8-persistence

## 오늘의 핵심

- Checkpointer: 스레드별 상태 저장, 타임 트래블(과거 시점 복구/조회) 가능
- Memory Store: 스레드를 넘어선 장기 기억(사용자별 기억 공유)
- Semantic Search: 임베딩을 활용한 의미 기반 기억 검색
